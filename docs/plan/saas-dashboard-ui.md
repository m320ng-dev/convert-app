# SaaS 대시보드 UI 개편 계획

## 목표

홈 화면과 모든 변환 툴 페이지를 미니멀한 SaaS 대시보드형 UI로 통일한다.

## 범위

- 공통 툴 목록 데이터를 분리해 홈 카드와 사이드바가 같은 정보를 사용한다.
- 전역 앱 셸을 추가해 데스크톱 고정 사이드바와 모바일 드로어 사이드바를 제공한다.
- 현재 경로 기준으로 홈 또는 선택된 툴을 시각적으로 강조한다.
- 홈 화면은 대시보드형 요약과 툴 그리드로 재구성한다.
- 기존 각 툴의 변환 로직은 유지하고, 전역 폼/결과 영역 스타일로 SaaS 톤을 맞춘다.

## 구현 메모

- 기존 툴 페이지 내부 로직과 상태 관리는 변경하지 않는다.
- 새 파일은 공통 레이아웃과 툴 메타데이터에 한정한다.
- 검증은 `npm run lint`와 `npm run build`로 수행한다.
