# UI/UX 레이아웃 개편 계획

## 목표

- 홈과 모든 변환 도구 페이지에 동일한 앱 셸과 디자인 시스템을 적용한다.
- 공통 사이드바 메뉴를 도입해 어느 화면에서든 다른 도구로 이동할 수 있게 한다.
- 현재 선택된 도구 또는 홈 화면이 메뉴에서 시각적으로 드러나게 한다.
- 모바일에서는 사이드바가 상단 탐색과 가로 스크롤 메뉴로 자연스럽게 축소되도록 한다.

## 구현 범위

- 도구 목록 메타데이터를 공통 모듈로 분리한다.
- 루트 레이아웃에 공통 앱 셸을 적용한다.
- 홈 화면을 도구 탐색 중심의 대시보드형 레이아웃으로 정리한다.
- 기존 도구 페이지의 기능 로직은 유지하고, 바깥 여백과 제목/패널/입력 영역 스타일을 통일한다.

## 검증

- `npm run lint`
- `npm run build`
